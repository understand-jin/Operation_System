# 시뮬레이션 결과 요약표 계산 로직

본 문서는 시뮬레이션 하단에 표시되는 **[시뮬레이션 결과 요약]** 표의 각 컬럼 및 행이 어떻게 계산되는지 설명합니다.

---

## 1. 주요 지표(KPI) 계산 공식

표의 왼쪽 영역에 표시되는 지표들은 각 자재(Material) 데이터를 카테고리별로 합산하여 산출합니다.

| 컬럼명 | 계산 공식 및 의미 | 코드 위치 |
| :--- | :--- | :--- |
| **원가** | 개별 자재의 `기말 재고 금액`을 모두 더한 값 | Line 823 |
| **출하원가** | 자재별 예산(한 달 판매 실적 기반)의 총합 | Line 823 |
| **출하판가** | 자재별 예상 판매가의 총합 | Line 823 |
| **회전월** | `원가 ÷ 출하원가`<br>*(현재 재고를 다 파는 데 걸리는 예상 개월 수)* | Line 828 |

---

## 2. 부진재고량 및 합계 계산

표의 오른쪽 영역은 시뮬레이션 결과로 도출된 **부진재고(시효임박)** 데이터를 보여줍니다.

### ① 분기별 데이터 (26년 1Q, 2Q...)
*   시뮬레이션 과정에서 유효기간 마지노선(6개월 전)에 도달했을 때 남아있는 재고 수량을 해당 분기 칸에 합산합니다.
*   **코드**: `pivot_table(..., aggfunc="sum")` (Line 789)

### ② 합계 (가로 합계)
*   특정 행(소분류/소계/총계)에서 발생한 **모든 분기의 부진재고량을 가로로 전부 더한 값**입니다.
*   **코드**: `pivot_detail["합계"] = pivot_detail.sum(axis=1)` (Line 792)

---

## 3. 행별 집계 기준

| 행 구분 | 집계 방식 |
| :--- | :--- |
| **총계 (Grand Total)** | 전체 데이터셋의 모든 수식 결과를 하나로 합산 (표의 맨 윗줄) |
| **소계 (Sub Total)** | 특정 '대분류'에 속한 모든 자재들의 수치를 합산 |
| **일반 행 (소분류)** | '대분류 + 소분류' 조합으로 그룹화된 상세 데이터 |

---

## 4. 특이 사항 (출하원가 0원 자재)
*   출하원가가 0원인 자재는 `회전월` 계산 시 분모가 0이 되므로 **0.0**으로 표시됩니다.
*   이러한 자재는 시뮬레이션 중 재고가 전혀 차감되지 않으므로, 유효기간 도래 시 **기초 재고 전체가 부진재고량으로 집계**되어 합계에 반영됩니다.
